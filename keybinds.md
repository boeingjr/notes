Change to Emacs keybinds, then do
~~~
C-x c         find class
C-x r         recent files
C-x q         recent changes
C-x ...       <whatever you dream of>

C-m m         "Move Caret to Matching Brace"
C-m z         previous edit location
C-m C-z       next edit location
C-m ...       <can be made true>

C-c r         Refactor rename this
C-c f         Refactor rename file
C-c t         
C-c t         surround with [], (), etc
C-c ...       <give command>

M-p           bw block    
M-n           fw block
C-+           increase font
C--           decrease font
C-æ           fold in
C-ø           fold out (stand in block)
C-S-æ         fold all in
C-S-ø         fold all out

C-M-s   DEF open settings

see
https://medium.com/mindorks/11-android-studio-shortcuts-every-android-developer-must-know-a153e736e611
